## Q3: genReqId 怎么变成 request.id

**Context**: `apps/api/src/server.ts:17`

**Key points**:

- 启动时 `genReqId` 经 `reqIdGenFactory` 挂到 Fastify 实例
- 每个请求进入 `routeHandler` 时调用 `getGenReqId(server, rawReq)` 生成 id
- 构造 `Request` 时把 id 写入 `this.id`，handler 里即 `request.id`

**Details**: [../knowledge/fastify/gen-req-id-to-request-id.md](../knowledge/fastify/gen-req-id-to-request-id.md)
