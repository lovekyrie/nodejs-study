# Learning Journal: 2026-05-24

## Q1: 检查 Docker 是否可用和启动项目依赖的命令分别做什么？

**Context**: `compose.yaml`、`docs/deployment.md`

**Key points**:

- `docker --version` 或 `docker version` 用于确认 Docker CLI，后者还可检查是否能连接 Docker Engine。
- `docker compose version` 用于确认 Compose 子命令可用，仍不会启动 PostgreSQL 或 Redis。
- `docker compose up -d postgres redis` 会根据 `compose.yaml` 创建并后台启动本项目需要的 PostgreSQL 与 Redis 服务。
- `docker build -f apps/api/Dockerfile .` 会构建 API 容器镜像，用于验证可部署构建，而不是启动数据库。
- “Docker 可用”、“项目依赖已运行”和“应用镜像可构建”是三个不同的验证层次。
